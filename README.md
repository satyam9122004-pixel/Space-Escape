# Space-Escape

Space Evader is a browser-based survival game built with HTML5, CSS3, and JavaScript. Pilot a ship to dodge dynamically spawning asteroids of varying sizes. It features real-time collision detection, immersive audio, and cinematic effects for a fast-paced arcade experience.
