Space Evader 🚀
A fast-paced, browser-based survival game where you pilot a spaceship through a hazardous asteroid field. Avoid the asteroids for as long as possible to survive!

🎮 How to Play
Start the Game: Click the "Start" button to launch your ship.

Move: Use your mouse/cursor to move the ship around the game area. The ship follows your cursor in real-time.

Objective: Avoid colliding with the falling asteroids.

Game Over: If an asteroid hits your ship, it's game over! You'll see a blast effect and can click "Restart" to try again.

✨ Features
Smooth Controls: Real-time mouse tracking for precise ship movement.

Dynamic Obstacles: Asteroids of 7 different sizes and textures spawn at random intervals and positions.

Immersive Audio: Includes launch sounds, engine movement loops, and explosion effects upon collision.

Visual Effects: Features a "blast" video overlay when the ship is destroyed.

Responsive Logic: Collision detection ensures a fair and challenging gameplay experience.

🛠️ Built With
HTML5: Structure and multimedia (Video/Audio) integration.

CSS3: Game styling, space background, and layout.

JavaScript (Vanilla): Game logic, interval management, and collision detection.
